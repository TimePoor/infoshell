# 개발 명령어

## 시스템 환경
- OS: Windows
- 쉘: pwsh (PowerShell 사용 금지, pwsh만 사용)
- 인코딩: UTF-8 주의

## 패키지 관리
```bash
npm install          # 의존성 설치
npm run dev          # 개발 모드 실행
npm run build        # 프로덕션 빌드
npm run package      # 앱 패키징
```

## 주의사항
- PowerShell 사용 금지, pwsh만 사용
- 파일 일괄 수정 절대 금지
- 배포 전 허락 필수
- 인코딩 항상 UTF-8 확인
